1-Télécharger Power BI Desktop sous le lien suivant: 
	https://powerbi.microsoft.com/fr-fr/desktop/ 
2-Avant d'utiliser l'application nommée 'Dashbord_BI.pbix',
de préférence d'utiliser une connexion internet
afin de pouvoir visualiser les différentes cartes 
géographiques comme : 'ArcGIS Map' qui se trouve 
dans la page "Performances de l'Algérie à l'export".
Ces cartes renseignent l'utilisateur final sur 
les informations relatives à un sujet d'analyse
determiné en utilisant des projections de ces informations
sur chaque région/pays. 
3-Désormais, vous pouvez lancer le fichier "Dashbord_BI.pbix".
